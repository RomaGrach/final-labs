# Final Labs

Сборник лабораторных работ по компьютерным сетям для студентов. В каждой папке находится основной текст лабораторной в `Markdown`, а также её версии в `Word` и `PDF`.

## Что внутри

- `LR1-GNS3` - первый стенд, ARP и ICMP.
- `LR2-Ethernet-Hub-Switch` - сравнение общей среды и коммутации.
- `LR3-VLAN-802.1Q` - VLAN и trunk 802.1Q.
- `LR4-STP-BPDU` - петли L2, STP и BPDU.
- `LR5-WiFi-802.11` - анализ кадров 802.11 и RTS/CTS.
- `LR6-IP-Subnets` - адресация, подсети и локальность ARP.
- `LR7-Routing-TTL-Fragmentation` - статическая маршрутизация, TTL, фрагментация и DF.

## Структура каждой лабораторной

В каждой папке лабораторной находятся:

- `README.md` - полный Markdown-отчёт.
- `report.docx` - оформленный отчёт в Word.
- `report.pdf` - экспортированный PDF.

Для первой лабораторной дополнительно есть папка `Ресурсы` с изображениями, которые используются внутри `README.md`.
