# Final Labs

Самодостаточный репозиторий с финальными вариантами лабораторных работ по сетям. Внутри находятся только локальные материалы: Markdown-отчёты, `DOCX`, `PDF`, файлы захвата, схемы, конфигурации, проекты и скриншоты.

## Что внутри

- `LR1-GNS3` - первый стенд, ARP и ICMP.
- `LR2-Ethernet-Hub-Switch` - сравнение общей среды и коммутации.
- `LR3-VLAN-802.1Q` - VLAN и trunk 802.1Q.
- `LR4-STP-BPDU` - петли L2, STP и BPDU.
- `LR5-WiFi-802.11` - анализ кадров 802.11 и RTS/CTS.
- `LR6-IP-Subnets` - адресация, подсети и локальность ARP.
- `LR7-Routing-TTL-Fragmentation` - статическая маршрутизация, TTL, фрагментация и DF.
- `Подготовка_к_защите.md` - краткие тезисы для повторения перед защитой.

## Структура каждой лабораторной

Обычно в каталоге лабораторной находятся:

- `README.md` - полный Markdown-отчёт.
- `report.docx` - оформленный отчёт в Word.
- `report.pdf` - экспортированный PDF.
- `assignment.md` - краткая формулировка задания.
- `screenshots/` - локальные изображения и подтверждающие скриншоты.
- `pcap/` - файлы захвата для Wireshark.
- `configs/` - конфигурации и команды.
- `gns3/` - схема стенда или связанные материалы GNS3.
